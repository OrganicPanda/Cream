Custom [angular-seed](https://github.com/angular/angularfire-seed) .

# Install

```
npm install
```

# Run

```
npm start
```

Now browse to the app at `http://localhost:8000/app/`.

# Tests

## Unit

```
npm test
npm run test-single-run
```

## End to End

```
npm start
npm run update-webdriver
npm run protractor
```
